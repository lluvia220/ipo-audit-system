# IPO审核动态查询系统

这是一个全面的IPO审核动态查询系统，包含以下主要功能：

## 功能模块

### 1. 月度专项报告
- 文字深度分析
- 图表辅助展示
- 趋势分析

### 2. 周报系统
- 关键词统计
- 图表为主展示
- 简单文字说明

### 3. 真实案例数据库
- 检索功能
- 详细案例分析
- 数据筛选

## 技术栈
- 前端：Vue.js + Element UI
- 后端：Node.js + Express
- 数据库：SQLite
- 图表：ECharts
- 文档：Markdown

## 目录结构
```
ipo-audit-system/
├── frontend/          # 前端Vue应用
├── backend/           # 后端API服务
├── database/          # 数据库文件
├── reports/           # 报告模板和数据
├── cases/             # 案例数据库
└── docs/              # 文档
```
