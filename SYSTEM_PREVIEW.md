## 🛠️ 技术架构

### 前端技术
- **框架**: Vue.js 2.6
- **UI库**: Element UI
- **图表**: ECharts 5.4
- **样式**: 自定义CSS + 响应式设计

### 后端技术
- **框架**: Express.js + Flask
- **数据库**: SQLite
- **API**: RESTful API
- **跨域**: CORS支持

### 数据库结构
```sql
-- IPO案例表
ipo_cases (id, company_name, stock_code, industry, 
          application_date, review_status, result, 
          rejection_reason, audit_time, key_words, detailed_info)

-- 月度报告表  
monthly_reports (id, year_month, total_applications, 
                approved, rejected, withdrawn, key_trends, analysis_text)

-- 周报表
weekly_reports (id, week_number, year_week, applications_count, 
               keyword_stats, summary_text)
```
